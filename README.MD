## Langkah-langkah

### 1. Clone Repositori

```bash
git clone https://github.com/tamabii/bot-sonic-sol.git
```

```
cd bot-sonic-sol
```

### 2. Instal Dependensi

```bash
npm install
```

### 3. Konfigurasi .env File

buat file .env DI FOLDER

```bash
PRIVATE_KEYS=["input pk nya"]
```

### 4. Run script

```bash
npm run start
```

\*pastikan udah install node js
